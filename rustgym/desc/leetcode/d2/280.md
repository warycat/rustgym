<div><p>Given an unsorted array <code>nums</code>, reorder it <b>in-place</b> such that <code>nums[0] &lt;= nums[1] &gt;= nums[2] &lt;= nums[3]...</code>.</p>

<p><b>Example:</b></p>

<pre><b>Input:</b> <code>nums = [3,5,2,1,6,4]</code>
<b>Output:</b> One possible answer is [3,5,1,6,2,4]</pre>
